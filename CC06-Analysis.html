Lesson 6: Data Analysis

In this lesson, we will explore various data analysis techniques commonly employed in geology. These techniques play a pivotal role in extracting meaningful insights and drawing conclusions from geological datasets.

Descriptive Statistics

Descriptive statistics serve as the foundation for understanding and summarizing geological data. They provide valuable insights into key characteristics of geological datasets. Let's explore these statistics with geological examples:

# Descriptive statistics with geological data
# Sample data: Mineral hardness (Mohs scale)
mineral_hardness <- c(7, 6, 5, 8, 6, 7, 6, 9, 8, 5)

# Mean mineral hardness
mean_hardness <- mean(mineral_hardness)

# Median mineral hardness
median_hardness <- median(mineral_hardness)

# Standard deviation of mineral hardness
sd_hardness <- sd(mineral_hardness)

# Range of mineral hardness
range_hardness <- range(mineral_hardness)

# Percentiles of mineral hardness (25th and 75th percentiles)
percentiles_hardness <- quantile(mineral_hardness, c(0.25, 0.75))

In this example, we have mineral hardness data (on the Mohs scale), and we calculate the mean, median, standard deviation, range, and percentiles to summarize this geological attribute.

Linear Regression

Linear regression is a valuable tool for modeling relationships between variables. Let's apply it to geological data:

# Linear regression with geological data
# Sample data: Depth vs. Temperature in a geological borehole
depth <- c(100, 200, 300, 400, 500)
temperature <- c(20, 18, 15, 12, 10)

# Perform linear regression
regression_model <- lm(temperature ~ depth)

# Summary of the linear regression model
summary(regression_model)

In this example, we use linear regression to model the relationship between depth and temperature in a geological borehole.

Non-linear Regression

Geological phenomena often exhibit non-linear behavior. Let's apply non-linear regression to geological data: 

# Non-linear regression with geological data
# Sample data: Radioactive decay of a geological element
time <- c(0, 1, 2, 3, 4, 5)
concentration <- c(100, 75, 56, 42, 31, 23)

# Define a non-linear model (exponential decay)
non_linear_model <- nls(concentration ~ a * exp(b * time), start = list(a = 100, b = -0.2))

# Summary of the non-linear regression model
summary(non_linear_model)

In this example, we use non-linear regression to model the radioactive decay of a geological element, which often follows an exponential decay pattern.

Hypothesis Testing

Let's consider a geological hypothesis test involving the average porosity of two rock types, sandstone, and shale. The hypothesis is whether there is a significant difference in porosity between these two rock types.

# Hypothesis testing with geological data
# Sample data: Porosity (%) in Sandstone and Shale samples
sandstone_porosity <- c(15.2, 14.8, 15.5, 14.9, 15.1)
shale_porosity <- c(12.0, 11.8, 11.5, 12.2, 12.1)

# Independent two-sample t-test
t_test_result <- t.test(sandstone_porosity, shale_porosity)

# Check the p-value for significance
if (t_test_result$p.value < 0.05) {
  hypothesis_result <- "Reject Null Hypothesis (Significant Difference)"
} else {
  hypothesis_result <- "Fail to Reject Null Hypothesis (No Significant Difference)"
}

# Print the hypothesis test result
cat("Hypothesis Test Result:", hypothesis_result, "\n")

In this example, we perform an independent two-sample t-test to determine if there is a significant difference in porosity between sandstone and shale samples. The hypothesis test result will indicate whether the difference in porosity is statistically significant or not.

Interpreting the results of a hypothesis test, such as a t-test, involves assessing the statistical evidence to determine whether your hypothesis is supported or not. Here's how to interpret the results:

  1. Null Hypothesis (H0): This is the default assumption that there is no significant difference or effect. In the context of a t-test, it typically suggests that there is no difference between the groups being compared.
  2. Alternative Hypothesis (Ha): This is the statement you want to test. It suggests that there is a significant difference or effect between the groups.
  3. P-value: The p-value represents the probability of obtaining results as extreme as, or more extreme than, the observed data if the null hypothesis is true. In other words, it quantifies the strength of the evidence against the null hypothesis.
    • A small p-value (usually less than 0.05) suggests that the observed results are unlikely to have occurred by chance alone, indicating evidence against the null hypothesis.
    • A large p-value (greater than 0.05) suggests that the observed results could plausibly occur by chance, providing weaker evidence against the null hypothesis.
  4. Significance Level (Alpha): This is the predetermined threshold (typically set at 0.05) that determines whether the p-value is considered statistically significant. If the p-value is less than alpha, you reject the null hypothesis.

Interpreting the Results:

  • If p-value < Alpha (e.g., p < 0.05): You can conclude that there is strong evidence against the null hypothesis. You reject the null hypothesis in favor of the alternative hypothesis. You may state that there is a statistically significant difference or effect.
  • If p-value ≥ Alpha (e.g., p ≥ 0.05): You do not have enough evidence to reject the null hypothesis. You cannot conclude that there is a significant difference or effect. The results are not statistically significant.

In the context of geological hypothesis testing, here are a few examples:

If you are testing whether the porosity of sandstone is significantly different from shale and you get a p-value of 0.03 (p < 0.05), you would conclude that there is strong evidence that the porosity of the two rock types differs significantly.

If you are testing whether there is a significant difference in compressive strength among granite, basalt, and limestone and you get a p-value of 0.25 (p ≥ 0.05), you would fail to find significant evidence of a difference in compressive strength among these geological formations.

Always consider the context and the specific hypotheses you are testing in geological research when interpreting the results of hypothesis tests. The p-value provides a quantitative measure of the evidence against the null hypothesis, but the interpretation should be based on both statistical significance and practical significance in your field of study.

ANOVA (Analysis of Variance)

Let's consider an ANOVA analysis involving the compressive strength of rock samples from three different geological formations: Granite, Basalt, and Limestone.

# ANOVA with geological data
# Sample data: Compressive Strength (MPa) in different geological formations
granite_strength <- c(120, 125, 130, 122, 128)
basalt_strength <- c(90, 88, 92, 91, 89)
limestone_strength <- c(70, 72, 68, 71, 73)

# Perform one-way ANOVA to compare compressive strength across formations
anova_result <- aov(c(granite_strength, basalt_strength, limestone_strength) ~ 
                     rep(c("Granite", "Basalt", "Limestone"), each = 5))

# Check for significant differences
if (summary(anova_result)$Pr[1] < 0.05) {
  anova_result_summary <- "Reject Null Hypothesis (Significant Difference)"
} else {
  anova_result_summary <- "Fail to Reject Null Hypothesis (No Significant Difference)"
}

# Print the ANOVA result summary
cat("ANOVA Result Summary:", anova_result_summary, "\n")

In this example, we use one-way ANOVA to compare the compressive strength of rock samples from different geological formations. The ANOVA result summary will indicate whether there are significant differences in compressive strength among the formations.

Suppose you perform an ANOVA to test whether there is a significant difference in the compressive strength of rock samples from three different geological formations: Granite, Basalt, and Limestone. If the p-value is 0.02 (p < 0.05), you would conclude that there is strong evidence that at least one pair of formations has a significantly different compressive strength. You would then perform post-hoc tests (e.g., Tukey's HSD) to determine which specific pairs of formations are different from each other.
