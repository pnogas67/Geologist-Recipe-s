Lesson 2: R basics

In this lesson, we'll cover the foundational aspects of working with R, including its unique syntax, essential data structures, variable assignment, and data types.

Syntax and Data Structures

R employs a distinctive syntax and provides several fundamental data structures:

  • Vectors: are one-dimensional arrays that can contain elements of the same data type, such as numbers, characters, or logical values. They are the most basic data structure in R and are used extensively for calculations and operations. You can create a vector like this:
    • my_vector <- c(1, 2, 3, 4, 5)

  • Matrices: are two-dimensional arrays with rows and columns. They are ideal for performing matrix algebra and representing structured data. To create a matrix, you can use the matrix() function:
    • my_matrix <- matrix(1:9, nrow = 3, ncol = 3)

  • Data Frames: are similar to tables in a database or spreadsheets. They are used for storing data in rows and columns, where each column can have a different data type. Data frames are essential for working with structured datasets. You can create a data frame like this:

    • my_data_frame <- data.frame(Mineral = c("Quartz", "Calcite", "Feldspar"),
                             Hardness = c(7, 3, 6))

  • Lists: are versatile data structures in R that can hold elements of different data types. Lists are often used to store collections of objects, including vectors and data frames. To create a list:

    • my_list <- list(density = c(3.2, 2.7, 4.3), mineral = "Pyrite", sulphide = TRUE)

    Variable Assignment and Data Types

    Variable assignment in R is straightforward. You can assign values to variables using either the <- operator or =. Here are some examples: 
    x <- 10
name <- "Quartz"
    R supports various data types, including:
    • Numeric: For representing numbers (e.g., 3.14, 42).
    • Character: For text and strings (e.g., "Hello, World!").
    • Logical: For boolean values (TRUE or FALSE).
    • Factor: For categorical data, which is useful in statistical analyses.

    Basic Operations in R

    R provides a wide range of basic operations for arithmetic, statistical, and mathematical calculations. You can perform standard arithmetic operations like addition, subtraction, multiplication, and division. Here's an example: 
    a <- 5
b <- 3
result <- a + b  # result will be 8
    These fundamental concepts serve as the building blocks for more advanced data manipulation and analysis in R. In the upcoming chapters, we'll explore these concepts further and delve into more complex data operations and analysis techniques.
