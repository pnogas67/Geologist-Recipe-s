Lesson 3: Data Import and Export

In this lesson, we'll explore how to work with data in R by learning how to import data from various file formats and export data to different formats.

Reading Data from CSV, Excel, and Other File Formats

R provides numerous functions and packages for reading data from various file formats. Some commonly used methods include:

  1. CSV Files: Comma-separated values (CSV) files are one of the most common data formats. You can read CSV files using the read.csv() function: 
    2. my_data <- read.csv("my_data.csv")
  2. Excel Files: To read data from Excel files, you can use packages like readxl or openxlsx. For example: 
    3. library(readxl)
my_data <- read_excel("my_data.xlsx")
  3. Other Formats: R supports many other formats, such as JSON, XML, and databases. Packages like jsonlite and XML provide functions for reading data from these formats.

Exporting Data to Various Formats

Exporting data from R is just as important as importing it. You can export data to formats like CSV, Excel, and more using functions like:

  1. CSV Files: To export data to a CSV file, you can use the write.csv() function: 
    2. write.csv(my_data, "exported_data.csv")
  2. Excel Files: For exporting data to Excel files, you can use packages like writexl or openxlsx. Here's an example using writexl: 
    3. library(writexl)
write_xlsx(my_data, "exported_data.xlsx")
  3. Other Formats: Depending on your needs, you can export data to formats like JSON, XML, or databases. Packages like jsonlite and XML can be helpful for these tasks.

In this chapter, you've learned the basics of importing and exporting data in R. These skills are crucial for working with real-world datasets and integrating your data analysis workflow with external tools and systems.
